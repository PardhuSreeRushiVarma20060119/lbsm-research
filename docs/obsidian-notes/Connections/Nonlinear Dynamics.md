# Nonlinear Dynamics

LBSM behavioral regimes map to classical nonlinear dynamics concepts:

- **Stable** → fixed point attractor
- **Exploratory** → limit cycle
- **Adaptive** → heteroclinic orbit / slow manifold
- **Unstable** → chaotic repeller / strange attractor

## Relevant Concepts

- Lyapunov stability — stable regime = Lyapunov-stable equilibrium
- Bifurcation — transition to unstable = bifurcation event
- Manifold theory — behavioral manifolds = invariant manifolds
